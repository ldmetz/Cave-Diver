# Cave Diver Game

A Java application that generates a cave and searches for an escape route.

## Run Locally

Requires JDK 21+

### Clone repository:
```bash
  git clone https://github.com/ldmetz/Cave-Diver.git
```

### Build: 
```bash
  cd Cave-Diver
  javac -d out src/*.java
```

### Run: 
```bash
  java -cp out CaveDiverFrame
```
